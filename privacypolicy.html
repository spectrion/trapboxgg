Privacy Policy

How Trapbox processes and protects security data

Overview

Trapbox is a Discord security system designed to detect, prevent, and respond to raids, abuse, and malicious activity using automated threat intelligence and cross-server protection.

Data We Collect

  • Discord User IDs
  • Discord Server IDs
  • Moderation actions (ban/unban logs)
  • Incident reports submitted by servers
  • Honeypot interaction events (if triggered)

How We Use Data

  • Detect and prevent raids or malicious activity
  • Maintain a shared threat intelligence network
  • Automate moderation actions across connected servers
  • Improve system accuracy and security protection

Security Network & Logging

By using Trapbox or participating in servers that use Trapbox, you acknowledge that certain security-related data (including Discord User IDs, Server IDs, and moderation events) may be logged as part of a shared protection system.

This information contributes to a cross-server security intelligence network used to detect and prevent raids, abuse, and coordinated attacks.

Data Retention

Trapbox may store data temporarily or long-term depending on its relevance to security and moderation purposes. Data may be periodically reviewed and removed when no longer required for threat prevention or system integrity.

Data Sharing

  • No sale of user data
  • Shared only between Trapbox-enabled servers
  • May be disclosed if required by law

Security

We use reasonable safeguards to protect stored data, but no system is completely secure.

User Rights

Depending on your region, you may request review or correction of stored data through server administrators or the Trapbox system dashboard.

Third-Party Services

Trapbox may rely on Discord and infrastructure providers. Each service has its own privacy policy.

Changes

This Privacy Policy may be updated over time. Continued use of Trapbox indicates acceptance of changes.
